> **제출 코드**
> 
- 사용 알고리즘 : `DP`

마지막에 한칸 또는 두칸이 오는 경우를 더해주면서 계산하는 DP 문제

```java
class Solution {
    public long solution(int n) {
        long[] dp = new long[n+1];
        dp[1] = 1;
        if(n == 1) return 1;
        dp[2] = 2;  
        for(int i=3; i<=n; i++) dp[i] = (dp[i-1] + dp[i-2])%1234567;
        return dp[n];
    }
}
```
